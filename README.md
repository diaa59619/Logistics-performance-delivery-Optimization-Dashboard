# Logistics-performance-delivery-Optimization-Dashboard
Logistics Performance & Profitability Analysis
📌 Project Overview

This project analyzes the operational and financial performance of a logistics company using Python and Tableau.

The goal was to evaluate:

Revenue & Profit performance

On-Time Delivery rate

Shipping mode efficiency

Route-level revenue contribution

Monthly performance trends

This end-to-end project simulates a real-world business scenario.

📊 Tools & Technologies Used

Python (Pandas, NumPy)

Jupyter Notebook

Tableau

Data Visualization

KPI Analysis

📁 Dataset

25,000 simulated shipment records

Features include:

Shipping Mode

Origin City

Destination City

Revenue

Cost

Profit

Delivery Time

Delay Days

On-Time Flag

Month

Dataset was generated using Python for analytical demonstration.

📈 Key KPIs
KPI	Value
Total Revenue	₹73M+
Total Profit	₹20M+
On-Time Delivery Rate	~50%
Avg Delivery Time	~5.3 Days
📊 Dashboard Insights
1️⃣ Shipping Mode Analysis

Revenue distribution is balanced across Air, Rail, and Road.

Delay patterns vary slightly by shipping mode.

2️⃣ Operational Efficiency

On-Time rate indicates improvement opportunity.

Certain months show performance variation.

3️⃣ Route Contribution

Top routes contribute disproportionately to revenue.

City-level delay differences highlight operational bottlenecks.

📌 Business Recommendations

Improve operational monitoring for high-delay shipping modes.

Introduce SLA tracking for routes with frequent delays.

Focus on high-profit routes for expansion.

Implement performance tracking dashboards at monthly level.
